# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2025-05-10

### Added

- 增加ui界面，包括连接，设备列表，设置(通用，网络，显示，网络，关于)
- 增加切换中英语言，主题，最大渲染帧率，最大捕获帧率
- 增加EventBus增加线程池，支持动态扩容
- 增加远控功能，支持键鼠同步，音频同步
- 增加支持Pulse音频驱动，X11桌面驱动
